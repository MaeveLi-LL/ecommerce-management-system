# 前端样式修改教程

## 📚 什么是CSS样式？

CSS（层叠样式表）是用来美化网页的代码。就像给房子刷油漆一样，CSS给网页添加颜色、大小、位置等样式。

---

## 🎨 样式文件在哪里？

### 1. 全局样式文件
**位置**：`frontend/src/index.css`
- 这个文件控制整个网站的通用样式
- 比如字体、背景色、滚动条等

### 2. 页面样式文件
每个页面都有自己的样式文件：
- **登录页面**：`frontend/src/pages/Login.css`
- **注册页面**：`frontend/src/pages/Register.css`
- **Dashboard页面**：`frontend/src/pages/Dashboard.css`
- **商品管理页面**：`frontend/src/pages/Products.css`
- **分类管理页面**：`frontend/src/pages/Categories.css`

---

## 🖊️ 如何修改样式？

### 步骤1：找到要修改的样式文件

**例子**：想修改登录页面的背景颜色

1. 在Cursor左侧文件树中
2. 找到 `frontend` 文件夹
3. 展开 `src` 文件夹
4. 展开 `pages` 文件夹
5. 找到 `Login.css` 文件
6. 双击打开

### 步骤2：找到要修改的样式代码

**例子**：修改背景颜色

在 `Login.css` 文件中，找到这段代码：
```css
.login-background {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 步骤3：修改颜色值

**颜色代码说明**：
- `#667eea` = 蓝色
- `#764ba2` = 紫色
- `#ff0000` = 红色
- `#00ff00` = 绿色
- `#ffffff` = 白色
- `#000000` = 黑色

**修改方法**：
把 `#667eea` 改成你想要的颜色，比如改成红色 `#ff0000`

### 步骤4：保存文件

按 `Ctrl + S` 保存文件

### 步骤5：查看效果

刷新浏览器页面（按 `F5`），就能看到新的颜色了！

## 🎨 常用样式修改示例

### 1. 修改背景颜色

**找到这段代码**：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**改成单色背景**：
```css
background: #f0f0f0;  /* 浅灰色 */
```

**改成其他渐变色**：
```css
background: linear-gradient(135deg, #ff6b6b 0%, #ee5a6f 100%);  /* 红色渐变 */
```

### 2. 修改文字颜色

**找到这段代码**：
```css
color: #333;
```

**改成其他颜色**：
```css
color: #ff0000;  /* 红色文字 */
color: #0000ff;  /* 蓝色文字 */
```

### 3. 修改字体大小

**找到这段代码**：
```css
font-size: 28px;
```

**改成其他大小**：
```css
font-size: 32px;  /* 更大 */
font-size: 24px;  /* 更小 */
```

### 4. 修改按钮颜色

**找到这段代码**：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**改成单色**：
```css
background: #1890ff;  /* 蓝色按钮 */
```

### 5. 修改卡片圆角

**找到这段代码**：
```css
border-radius: 16px;
```

**改成其他圆角**：
```css
border-radius: 8px;   /* 小圆角 */
border-radius: 24px;  /* 大圆角 */
border-radius: 0;     /* 没有圆角（方形） */
```

## 📝 样式代码结构说明

### CSS代码的基本格式：

```css
/* 这是注释，不会影响样式 */

.样式名称 {
  属性名: 属性值;
  属性名: 属性值;
}
```

### 例子：

```css
.login-title {
  font-size: 28px;      /* 字体大小 */
  color: #333;          /* 文字颜色 */
  font-weight: bold;    /* 字体粗细 */
  text-align: center;   /* 文字对齐方式 */
}
```

---

## 🎯 常用属性说明

| 属性名 | 作用 | 例子 |
|--------|------|------|
| `color` | 文字颜色 | `color: #333;` |
| `background` | 背景颜色 | `background: #fff;` |
| `font-size` | 字体大小 | `font-size: 16px;` |
| `font-weight` | 字体粗细 | `font-weight: bold;` |
| `padding` | 内边距 | `padding: 20px;` |
| `margin` | 外边距 | `margin: 10px;` |
| `border-radius` | 圆角 | `border-radius: 8px;` |
| `box-shadow` | 阴影 | `box-shadow: 0 2px 8px rgba(0,0,0,0.1);` |

## 💡 修改技巧

### 技巧1：先备份
修改前，先复制原来的代码，这样改错了可以恢复

### 技巧2：小步修改
一次只改一个地方，改完保存看看效果

### 技巧3：使用浏览器开发者工具
1. 按 `F12` 打开开发者工具
2. 点击左上角的"选择元素"按钮
3. 点击页面上你想修改的元素
4. 在右侧可以看到这个元素的样式
5. 可以直接在浏览器里修改看效果（不会保存）

## ❓ 遇到问题？

如果修改后没有效果：
1. 检查是否保存了文件（`Ctrl + S`）
2. 检查代码是否有语法错误（比如少了分号 `;`）
3. 刷新浏览器（`F5` 或 `Ctrl + F5`）
4. 检查前端服务是否在运行
